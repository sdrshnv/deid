About De-Id

De-Id is a local text de-identification tool. Paste text containing personal information, and De-Id redacts emails, names, and file paths — all processing stays on your machine.

The app has a simple two-panel layout: paste your text on the left, click De-Id, and de-identified text appears on the right with PII replaced by [REDACTED-email], [REDACTED-name], or [REDACTED-file] placeholders. Click the output to copy it to your clipboard.

Email and file path detection use built-in pattern matching and are always available. For name detection, De-Id uses Ollama running locally with the qwen3:4b model. A status indicator in the app shows whether Ollama is connected. Without Ollama, the app works in regex-only mode (emails and file paths).
